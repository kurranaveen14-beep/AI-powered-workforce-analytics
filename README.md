# AI & Tech Workforce Intelligence Dashboard

A professional Power BI-style workforce analytics dashboard built using AI-powered tools and real-world workforce datasets containing 12,000+ rows and 23 columns.

## Project Overview

This project analyzes:

* Tech layoffs trends
* Hiring patterns
* AI adoption impact
* Employee sentiment
* Job security metrics
* Industry-wise workforce insights
* Market condition analysis

The dashboard was designed to simulate an executive-level Business Intelligence reporting system with modern visualization and data storytelling concepts.

## Key Features

* KPI-based executive dashboard
* Industry-wise layoffs analysis
* Top hiring roles visualization
* AI Adoption vs Job Security analysis
* Market condition distribution
* Workforce insights and business storytelling
* Professional Power BI-style dashboard design

## Dataset Information

* Rows: 12,000+
* Columns: 23
* Domain: Workforce Analytics / AI Impact / Hiring Trends
